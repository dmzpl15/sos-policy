# Privacy Policy for Global SOS Signal

**Last Updated: 2024.05.22**

(주)ArtSoft (이하 '회사')는 사용자의 개인정보를 소중하게 생각하며, 'Global SOS Signal' 앱(이하 '앱') 서비스 이용 시 개인정보 보호법을 준수합니다.

## 1. 수집하는 개인정보 (Information We Collect)
본 앱은 사용자의 긴급 구조를 위해 다음과 같은 정보를 수집하고 사용합니다.
* **위치 정보 (Location Data):** 긴급 SOS 활성화 시 사용자의 현재 위치(위도, 경도)를 파악하여 지정된 수신자에게 전송하기 위해 수집합니다.
* **연락처 정보 (Contact Information):** 사용자가 직접 등록한 비상 연락처 정보를 SMS 발송 대상으로 사용합니다.

## 2. 수집 목적 및 사용 (How We Use Information)
* **긴급 구조 지원:** 사용자가 SOS 버튼을 눌렀을 때, 현재 위치를 비상 연락처로 전송하고 긴급 전화를 연결하는 목적으로만 사용됩니다.
* **데이터 저장 안 함 (No Server Storage):** 수집된 위치 정보는 서버에 저장되지 않으며, 사용자가 기능을 종료하면 즉시 폐기됩니다.

## 3. 권한 사용 안내 (Permissions)
* **ACCESS_FINE_LOCATION / COARSE_LOCATION:** 정확한 위치 파악을 위해 필요합니다.
* **SEND_SMS:** 긴급 메시지 발송을 위해 필요합니다.
* **CALL_PHONE:** 112/911 등 긴급 전화 연결을 위해 필요합니다.

## 4. 제3자 제공 (Third-Party Sharing)
회사는 사용자의 개인정보를 외부에 제공하지 않습니다. 단, 사용자가 SOS 기능을 실행하여 직접 문자를 보낼 때 통신사를 통해 해당 정보가 전달됩니다.

## English Version Summary
This app collects **Location Data** to provide emergency services. This data is processed only on your device and is **NOT stored on any server**. When you activate SOS,
your coordinates are sent via SMS to your pre-registered emergency contacts. We request **Location, SMS, and Phone** permissions solely for these lifesaving features.
